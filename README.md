# Data_Science

Each folder has some excercises that I made in the [Python for Data Science and Machine Learning Bootcamp](https://www.udemy.com/python-for-data-science-and-machine-learning-bootcamp/learn/v4/overview)

## Requirements

1. Install [Python](https://www.python.org/downloads/)
2. Install [Conda](https://www.anaconda.com/download/)
    * Numpy --- conda install numpy ---
    * Pandas --- conda install pandas ---
    * Plotly --- conda install plotly ---
    * Cufflinks --- conda install cufflinks ---
3. Install [Jupyter notebook](http://jupyter.org/) and run --- jupyter notebook --- at the root of your dir/folder.
